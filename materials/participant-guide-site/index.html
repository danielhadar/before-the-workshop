חיבור

לגרום לקלוד לראות את הקבצים שלכם ב-Obsidian

  1. 1

    מצאו את קובץ ההגדרות

    פתחו Finder ← לחצו Cmd+Shift+G ← הקלידו:

    ~/Library/Application Support/Claude
  2. 2

    פתחו לעריכה

    מצאו קובץ בשם claude_desktop_config.json (או config.json) ופתחו אותו עם TextEdit (לחיצה ימנית ← Open With ← TextEdit)

  3. 3

    החליפו את התוכן

    מחקו הכל והדביקו את זה — שנו את הנתיב לנתיב ה-Vault האמיתי שלכם:

    { "mcpServers": { "filesystem": { "command": "npx", "args": [ "-y", "@modelcontextprotocol/server-filesystem", "/Users/yourname/Documents/yourname-vault" ] } } }
  4. 4

    שמרו והפעילו מחדש

    שמרו, סגרו את Claude לגמרי ופתחו מחדש

  1. 1

הכנת חומרים

אם הפרויקט שלכם הוא מותג אישי או תוכן: פוסטים מהסושיאל, מיילים ללקוחות, כתבות שפרסמתם. כל דבר שאתם גאים בו ומרגישים שמייצג את הסגנון שלכם.

אם הפרויקט שלכם הוא חברה או מותג: ספר מותג, מצגת משקיעים, הודעות לעיתונות. כל דבר שמדבר בשפה של המותג.
